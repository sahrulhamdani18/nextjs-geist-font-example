# Android Student Attendance and Grades App

This is a native Android application written in Java for managing student attendance and grades. Features include:
- Adding multiple subjects
- Adding multiple classes
- Automatic student data import from Excel files
- Attendance and grades input and management

The project includes source code and a generated APK file.

## Project Structure

- src/main/java: Java source code
- src/main/res/layout: XML layout files
- AndroidManifest.xml: App manifest
- build.gradle: Gradle build files

## Build and Run

Use Android Studio or Gradle command line to build the APK.
